# Project-2-Mortgage-Calculator
A simple mortgage calculator built with HTML, CSS, and JavaScript.

## Features

- Calculates monthly mortgage payments
- Supports 10, 15, and 30 year loans
- Input validation
- Responsive design
- CSS Grid layout
- Background gradient

## Formula

payment = (P * r) / (1 - (1 + r)^(-n))

Where:

- P = Principal
- r = Monthly interest rate
- n = Number of monthly payments
